# 📋 공통 API 명세

## 기본 정보

- **Base URL:** `https://{domain}/api`
- **Content-Type:** `application/json`
- **인증 방식:** JWT Bearer Token

---

## 공통 응답 형식

```json
{
  "success": true,
  "message": "API 호출 성공",
  "data": { }
}
```

| 필드 이름 | 타입 | 설명 |
| --- | --- | --- |
| success | Boolean | 요청 성공 여부 |
| message | String | 응답 메시지 |
| data | Object \| null | 응답 데이터 (실패 시 null) |

---

## 공통 에러 응답

### 400 Bad Request
```json
{
  "success": false,
  "message": "요청 형식이 올바르지 않습니다.",
  "data": null
}
```

### 401 Unauthorized
```json
{
  "success": false,
  "message": "인증 정보가 유효하지 않습니다. 다시 로그인해주세요.",
  "data": null
}
```

### 403 Forbidden
```json
{
  "success": false,
  "message": "권한이 없습니다.",
  "data": null
}
```

### 404 Not Found
```json
{
  "success": false,
  "message": "존재하지 않는 회원 정보입니다.",
  "data": null
}
```

### 500 Internal Server Error
```json
{
  "success": false,
  "message": "서버 내부 오류가 발생했습니다.",
  "data": null
}
```

## 공통 헤더

| 헤더 | 필수 여부 | 설명 |
| --- | --- | --- |
| `Authorization: Bearer {accessToken}` | 인증 필요 API | JWT AccessToken |
| `Content-Type: application/json` | Body 있는 API | JSON 형식 |
