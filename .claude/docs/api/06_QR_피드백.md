# 📷 QR 코드 & 피드백 API 명세

---

## 📍 22 QR 코드 이미지 생성 API

- **설명:** 특정 장소의 QR 웹페이지로 연결되는 QR 코드 이미지를 생성하여 반환하는 API입니다. 생성된 QR 코드를 스캔하면 해당 장소의 현재 온습도 조회 및 피드백 제출 페이지로 이동합니다. (ADMIN 권한 필요)
- **헤더:** `Authorization: Bearer {accessToken}`

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| placeId | 필수 | Int | QR 코드를 생성할 장소 ID |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Sample**

```
GET /api/qr?placeId=501
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | QR 코드 이미지 생성 완료 |
| 실패 | 401 | 인증 실패 |
| 실패 | 403 | ADMIN 권한 없음 |
| 실패 | 404 | 존재하지 않는 장소 |
| 실패 | 500 | 서버 내부 오류 |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| qrImageUrl | 필수 | String | 생성된 QR 코드 이미지 URL (PNG) |
| qrCode | 필수 | String | place 테이블의 qrCode 값 (URL 식별자) |
| webUrl | 필수 | String | QR 스캔 시 이동할 웹페이지 URL |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "API 호출 성공",
  "data": {
    "qrImageUrl": "https://reborn.kr/api/qr/image/a1b2c3d4.png",
    "qrCode": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "webUrl": "https://reborn.kr/place/a1b2c3d4-e5f6-7890-abcd-ef1234567890"
  }
}
```

**실패 케이스 - 401 Unauthorized**
```json
{ "success": false, "message": "인증 정보가 유효하지 않습니다. 다시 로그인해주세요.", "data": null }
```

**실패 케이스 - 403 Forbidden**
```json
{ "success": false, "message": "권한이 없습니다.", "data": null }
```

**실패 케이스 - 404 Not Found**
```json
{ "success": false, "message": "존재하지 않는 장소 정보입니다.", "data": null }
```

**실패 케이스 - 500 Internal Server Error**
```json
{ "success": false, "message": "서버 내부 오류가 발생했습니다.", "data": null }
```

## 📍 23 피드백 보내기 API

- **설명:** QR 웹페이지에서 방문자가 실내 환경에 대한 피드백을 제출하는 API입니다. 인증 없이 접근 가능하며, IP 주소와 세션 토큰으로 중복 제출을 방지합니다. 피드백 저장 후 해당 장소의 ADMIN 권한 사용자에게 FCM Push 알림이 비동기로 전송됩니다.
- **헤더:** 없음

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| qrCode | 필수 | String | place 테이블의 qrCode 값 (장소 식별자) |
| deviceId | 필수 | String | 피드백 대상 기기 ID |
| title | 필수 | String | 피드백 제목 (예: 너무 더워요, 춥습니다) |
| content | 선택 | String | 피드백 상세 내용 |
| sessionToken | 선택 | String | QR 접속 시 발급된 임시 세션 토큰 (중복 방지) |

**Request Sample**

```json
{
  "qrCode": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "deviceId": "arduino_room_01",
  "title": "너무 더워요",
  "content": "오후에 햇빛이 너무 강하게 들어와서 매우 덥습니다.",
  "sessionToken": "sess_7f3k2m9x"
}
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | 피드백 제출 완료 및 관리자 알림 발송 |
| 실패 | 400 | 필수 필드 누락 또는 잘못된 qrCode |
| 실패 | 404 | 존재하지 않는 장소 또는 기기 |
| 실패 | 429 | 동일 IP/세션 중복 제출 (Too Many Requests) |
| 실패 | 500 | 서버 내부 오류 |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| feedbackId | 필수 | Int | 생성된 피드백 고유 ID |
| status | 필수 | String | 피드백 처리 상태 (PENDING) |
| createdAt | 필수 | DateTime | 피드백 제출 시간 (ISO 8601) |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "피드백이 제출되었습니다.",
  "data": {
    "feedbackId": 1024,
    "status": "PENDING",
    "createdAt": "2026-05-17T21:05:04Z"
  }
}
```

**실패 케이스 - 400 Bad Request**
```json
{ "success": false, "message": "요청 형식이 올바르지 않습니다.", "data": null }
```

**실패 케이스 - 404 Not Found**
```json
{ "success": false, "message": "존재하지 않는 장소 정보입니다.", "data": null }
```

**실패 케이스 - 429 Too Many Requests**
```json
{ "success": false, "message": "이미 피드백을 제출하셨습니다. 잠시 후 다시 시도해주세요.", "data": null }
```

**실패 케이스 - 500 Internal Server Error**
```json
{ "success": false, "message": "서버 내부 오류가 발생했습니다.", "data": null }
```

## 📍 24 피드백 조회 API

- **설명:** 특정 장소 또는 기기에 접수된 피드백 목록을 조회하는 API입니다. 상태별 필터링(PENDING / APPROVED / REJECTED)을 지원하며, 관리자 앱의 피드백 관리 화면에서 사용됩니다. (ADMIN 권한 필요)
- **헤더:** `Authorization: Bearer {accessToken}`

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| placeId | 필수 | Int | 조회할 장소 ID |
| deviceId | 선택 | String | 특정 기기 피드백만 조회 |
| status | 선택 | String | 상태 필터 (PENDING / APPROVED / REJECTED) |
| page | 선택 | Int | 페이지 번호 (기본값: 0) |
| size | 선택 | Int | 페이지 크기 (기본값: 20) |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Sample**

```
GET /api/feedback?placeId=501&status=PENDING&page=0&size=20
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | 피드백 목록 조회 성공 |
| 실패 | 400 | placeId 누락 또는 잘못된 status 값 |
| 실패 | 401 | 인증 실패 |
| 실패 | 403 | ADMIN 권한 없음 |
| 실패 | 500 | 서버 내부 오류 |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| totalCount | 필수 | Int | 전체 피드백 수 |
| feedbacks | 필수 | Array | 피드백 목록 |
| feedbacks[].feedbackId | 필수 | Int | 피드백 고유 ID |
| feedbacks[].deviceId | 필수 | String | 피드백 대상 기기 ID |
| feedbacks[].deviceName | 필수 | String | 기기 이름 (방 이름) |
| feedbacks[].title | 필수 | String | 피드백 제목 |
| feedbacks[].content | 선택 | String | 피드백 내용 |
| feedbacks[].status | 필수 | String | 처리 상태 |
| feedbacks[].createdAt | 필수 | DateTime | 제출 시간 (ISO 8601) |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "API 호출 성공",
  "data": {
    "totalCount": 5,
    "feedbacks": [
      {
        "feedbackId": 1024,
        "deviceId": "arduino_room_01",
        "deviceName": "거실",
        "title": "너무 더워요",
        "content": "오후에 햇빛이 너무 강하게 들어와서 매우 덥습니다.",
        "status": "PENDING",
        "createdAt": "2026-05-17T21:05:04Z"
      }
    ]
  }
}
```

**실패 케이스 - 401 Unauthorized**
```json
{ "success": false, "message": "인증 정보가 유효하지 않습니다. 다시 로그인해주세요.", "data": null }
```

**실패 케이스 - 403 Forbidden**
```json
{ "success": false, "message": "권한이 없습니다.", "data": null }
```

**실패 케이스 - 500 Internal Server Error**
```json
{ "success": false, "message": "서버 내부 오류가 발생했습니다.", "data": null }
```

## 📍 25 피드백 개수 조회 API

- **설명:** 특정 장소의 피드백 개수를 상태별로 집계하여 반환하는 API입니다. 관리자 앱 대시보드의 피드백 현황 위젯에서 빠르게 미처리 피드백 수를 확인하는 데 사용됩니다. (ADMIN 권한 필요)
- **헤더:** `Authorization: Bearer {accessToken}`

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| placeId | 필수 | Int | 조회할 장소 ID |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Sample**

```
GET /api/feedback/count?placeId=501
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | 피드백 개수 조회 성공 |
| 실패 | 400 | placeId 누락 |
| 실패 | 401 | 인증 실패 |
| 실패 | 403 | ADMIN 권한 없음 |
| 실패 | 404 | 존재하지 않는 장소 |
| 실패 | 500 | 서버 내부 오류 |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| total | 필수 | Int | 전체 피드백 수 |
| pending | 필수 | Int | 미처리(PENDING) 피드백 수 |
| approved | 필수 | Int | 승인(APPROVED) 피드백 수 |
| rejected | 필수 | Int | 거절(REJECTED) 피드백 수 |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "API 호출 성공",

    "total": 42,
    "pending": 5,
    "approved": 31,
    "rejected": 6
  }
}
```
