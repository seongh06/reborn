# 📊 메트릭 API 명세

---

## 📍 12 장소 메트릭 데이터 전달 API

- **설명:** Arduino IoT 기기가 수집한 온도, 습도, 조도, 재실 인원 데이터를 서버로 전송하는 API입니다. 데이터는 metricLogs 테이블에 저장되며, 불쾌지수는 서버에서 자동 계산합니다. 기기 인증은 deviceId 기반으로 처리합니다.
- **헤더:** `X-Device-Id: {deviceId}`

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| temperature | 선택 | Decimal | 온도 (°C, 소수점 2자리) |
| humidity | 선택 | Decimal | 습도 (%, 소수점 2자리) |
| illuminance | 선택 | Int | 조도 (lux) |
| peopleCount | 선택 | Int | 재실 인원 수 |

**Request Sample**

```json
{
  "temperature": 26.50,
  "humidity": 62.30,
  "illuminance": 480,
  "peopleCount": 3
}
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | 메트릭 데이터 저장 완료 |
| 실패 | 400 | 요청 형식 오류 또는 모든 필드 null |
| 실패 | 401 | 유효하지 않은 deviceId |
| 실패 | 404 | 등록되지 않은 기기 |
| 실패 | 500 | 서버 내부 오류 |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| logId | 필수 | Long | 저장된 로그의 고유 ID |
| discomfort | 선택 | Decimal | 서버에서 계산된 불쾌지수 |
| createdAt | 필수 | DateTime | 데이터 저장 시간 (ISO 8601) |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "API 호출 성공",
  "data": {
    "logId": 10023,
    "discomfort": 74.21,
    "createdAt": "2026-05-17T21:05:04.123456Z"
  }
}
```

**실패 케이스 - 401 Unauthorized**
```json
{ "success": false, "message": "등록되지 않은 기기입니다.", "data": null }
```

**실패 케이스 - 500 Internal Server Error**
```json
{ "success": false, "message": "서버 내부 오류가 발생했습니다.", "data": null }
```

## 📍 13 장소 메트릭 조회 API

- **설명:** 특정 기기의 가장 최근 메트릭 데이터를 단건으로 조회하는 API입니다. QR 웹페이지 및 관리자 앱 대시보드에서 현재 실내 환경 상태를 표시하는 데 사용됩니다. 인증 없이 접근 가능합니다.
- **헤더:** 없음

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| deviceId | 필수 | String | 조회할 기기의 고유 ID |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Sample**

```
GET /api/metric/current?deviceId=arduino_room_01
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | 최신 메트릭 데이터 조회 성공 |
| 실패 | 400 | deviceId 누락 |
| 실패 | 404 | 등록되지 않은 기기 또는 수집 데이터 없음 |
| 실패 | 500 | 서버 내부 오류 |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| deviceId | 필수 | String | 기기 고유 ID |
| deviceName | 필수 | String | 기기 이름 (방 이름) |
| temperature | 선택 | Decimal | 온도 (°C) |
| humidity | 선택 | Decimal | 습도 (%) |
| illuminance | 선택 | Int | 조도 (lux) |
| peopleCount | 선택 | Int | 재실 인원 수 |
| discomfort | 선택 | Decimal | 불쾌지수 |
| createdAt | 필수 | DateTime | 데이터 수집 시간 (ISO 8601) |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "API 호출 성공",
  "data": {
    "deviceId": "arduino_room_01",
    "deviceName": "거실",
    "temperature": 26.50,
    "humidity": 62.30,
    "illuminance": 480,
    "peopleCount": 3,
    "discomfort": 74.21,
    "createdAt": "2026-05-17T21:05:04.123456Z"
  }
}
```

**실패 케이스 - 404 Not Found**
```json
{ "success": false, "message": "등록되지 않은 기기이거나 수집된 데이터가 없습니다.", "data": null }
```

**실패 케이스 - 500 Internal Server Error**
```json
{ "success": false, "message": "서버 내부 오류가 발생했습니다.", "data": null }
```

## 📍 13-1 메트릭 히스토리 조회 API

- **설명:** 특정 기기의 메트릭 로그 목록을 페이징하여 조회하는 API입니다. 관리자 앱의 센서 로그 히스토리 화면에서 사용되며, 해당 장소에 대한 접근 권한이 있는 사용자만 조회할 수 있습니다. (인증 필요)
- **헤더:** `Authorization: Bearer {accessToken}`

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| deviceId | 필수 | String | 조회할 기기의 고유 ID |
| page | 선택 | Int | 페이지 번호 (0부터 시작, 기본값 0) |
| size | 선택 | Int | 페이지 크기 (기본값 20) |
| sort | 선택 | String | 정렬 기준 (기본값 createdAt,desc) |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Sample**

```
GET /api/metric/history?deviceId=arduino_room_01&page=0&size=20
Authorization: Bearer eyJhbGciOi...
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | 조회 성공 — 로그 목록 및 페이징 정보 반환 |
| 실패 | 401 | 인증 실패 (유효하지 않거나 만료된 AccessToken) |
| 실패 | 403 | 해당 장소에 대한 접근 권한 없음 |
| 실패 | 404 | 등록되지 않은 기기 ID |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| deviceId | 필수 | String | 기기 고유 ID |
| logs | 필수 | List | 메트릭 로그 목록 |
| logs[].logId | 필수 | Long | 로그 ID |
| logs[].temperature | 선택 | Decimal | 온도 (°C) |
| logs[].humidity | 선택 | Decimal | 습도 (%) |
| logs[].illuminance | 선택 | Int | 조도 (lux) |
| logs[].peopleCount | 선택 | Int | 재실 인원 수 |
| logs[].createdAt | 필수 | DateTime | 데이터 수집 시간 (ISO 8601) |
| page | 필수 | Int | 현재 페이지 번호 |
| size | 필수 | Int | 페이지 크기 |
| totalElements | 필수 | Long | 전체 로그 개수 |
| totalPages | 필수 | Int | 전체 페이지 수 |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "API 호출 성공",
  "data": {
    "deviceId": "arduino_room_01",
    "logs": [
      {
        "logId": 102,
        "temperature": 26.50,
        "humidity": 62.30,
        "illuminance": 480,
        "peopleCount": 3,
        "createdAt": "2026-05-17T21:05:04.123456Z"
      }
    ],
    "page": 0,
    "size": 20,
    "totalElements": 1,
    "totalPages": 1
  }
}
```

**실패 케이스 - 403 Forbidden**
```json
{ "success": false, "message": "해당 장소에 대한 접근 권한이 없습니다.", "data": null }
```

**실패 케이스 - 404 Not Found**
```json
{ "success": false, "message": "등록되지 않은 기기입니다.", "data": null }
```

## 📍 14 통계 데이터 조회 API

- **설명:** 특정 기기의 메트릭 데이터를 기간별로 집계하여 통계 데이터를 반환하는 API입니다. 시간별, 일별, 주별 단위로 평균 온도, 습도, 불쾌지수를 조회할 수 있으며, 관리자 앱의 데이터 분석 화면에서 사용됩니다. (인증 필요)
- **헤더:** `Authorization: Bearer {accessToken}`

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| deviceId | 필수 | String | 조회할 기기의 고유 ID |
| unit | 필수 | String | 집계 단위 (HOUR: 시간별, DAY: 일별, WEEK: 주별) |
| from | 필수 | String | 조회 시작일 (ISO 8601, 예: 2026-05-01T00:00:00Z) |
| to | 필수 | String | 조회 종료일 (ISO 8601, 예: 2026-05-17T23:59:59Z) |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Sample**

```
GET /api/metric/stats?deviceId=arduino_room_01&unit=DAY&from=2026-05-01T00:00:00Z&to=2026-05-17T23:59:59Z
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | 통계 데이터 조회 성공 |
| 실패 | 400 | 필수 파라미터 누락 또는 잘못된 unit 값 |
| 실패 | 401 | 인증 실패 |
| 실패 | 403 | 해당 장소 접근 권한 없음 |
| 실패 | 404 | 등록되지 않은 기기 |
| 실패 | 500 | 서버 내부 오류 |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| deviceId | 필수 | String | 기기 고유 ID |
| unit | 필수 | String | 집계 단위 |
| stats | 필수 | Array | 집계 데이터 목록 |
| stats[].timestamp | 필수 | DateTime | 집계 기준 시간 |
| stats[].avgTemperature | 선택 | Decimal | 평균 온도 |
| stats[].avgHumidity | 선택 | Decimal | 평균 습도 |
| stats[].avgDiscomfort | 선택 | Decimal | 평균 불쾌지수 |
| stats[].avgPeopleCount | 선택 | Decimal | 평균 재실 인원 |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "API 호출 성공",
  "data": {
    "deviceId": "arduino_room_01",
    "unit": "DAY",
    "stats": [
      {
        "timestamp": "2026-05-01T00:00:00Z",
        "avgTemperature": 24.30,
        "avgHumidity": 58.10,
        "avgDiscomfort": 71.50,
        "avgPeopleCount": 2.3
      },
      {
        "timestamp": "2026-05-02T00:00:00Z",
        "avgTemperature": 25.10,
        "avgHumidity": 60.40,
        "avgDiscomfort": 72.80,
        "avgPeopleCount": 1.8
      }
    ]
  }
}
```

**실패 케이스 - 400 Bad Request**
```json
{ "success": false, "message": "요청 형식이 올바르지 않습니다.", "data": null }
```

**실패 케이스 - 401 Unauthorized**
```json
{ "success": false, "message": "인증 정보가 유효하지 않습니다. 다시 로그인해주세요.", "data": null }
```

**실패 케이스 - 403 Forbidden**
```json
{ "success": false, "message": "권한이 없습니다.", "data": null }
```

**실패 케이스 - 500 Internal Server Error**
```json
{ "success": false, "message": "서버 내부 오류가 발생했습니다.", "data": null }

## 📍 15 보고서 다운로드 API

- **설명:** 특정 기간 동안의 메트릭 데이터를 PDF 또는 CSV 형식의 보고서 파일로 생성하여 다운로드하는 API입니다. (ADMIN 권한 필요)
- **헤더:** `Authorization: Bearer {accessToken}`

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| placeId | 필수 | Int | 보고서를 생성할 장소 ID |
| from | 필수 | String | 조회 시작일 (ISO 8601) |
| to | 필수 | String | 조회 종료일 (ISO 8601) |
| format | 필수 | String | 파일 형식 (PDF / CSV) |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Sample**

```
GET /api/metric/report?placeId=501&from=2026-05-01T00:00:00Z&to=2026-05-17T23:59:59Z&format=PDF

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | 보고서 파일 생성 및 다운로드 URL 반환 |
| 실패 | 400 | 필수 파라미터 누락 또는 잘못된 format 값 |
| 실패 | 401 | 인증 실패 |
