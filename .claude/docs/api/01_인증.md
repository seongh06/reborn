## 📍 01 구글 로그인 API

- **설명:** Google OAuth 2.0을 통해 발급받은 ID Token을 서버에 전달하여 로그인 또는 회원가입을 처리하는 API입니다. 신규 사용자의 경우 자동으로 회원가입이 진행되며, 기존 사용자의 경우 로그인 처리 후 AccessToken과 RefreshToken을 발급합니다.
- **헤더:** 없음

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| idToken | 필수 | String | Google OAuth 2.0에서 발급받은 ID Token |

**Request Sample**

```json
{
  "idToken": "eyJhbGciOiJSUzI1NiIsImtpZCI6..."
}
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | 로그인 성공 및 토큰 발급 완료 |
| 실패 | 400 | idToken 누락 또는 형식 오류 |
| 실패 | 401 | 유효하지 않거나 만료된 Google ID Token |
| 실패 | 500 | 서버 내부 오류 |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| accessToken | 필수 | String | 발급된 JWT AccessToken |
| refreshToken | 필수 | String | 발급된 RefreshToken (Redis 저장) |
| userId | 필수 | String | 사용자 고유 ID |
| name | 필수 | String | 사용자 이름 |
| isNewUser | 필수 | Boolean | 신규 가입 여부 |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "API 호출 성공",
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiJ9...",
    "userId": "google_109234567890",
    "name": "홍길동",
    "isNewUser": false
  }
}
```

**실패 케이스 - 400 Bad Request**
```json
{ "success": false, "message": "요청 형식이 올바르지 않습니다.", "data": null }
```

**실패 케이스 - 401 Unauthorized**
```json
{ "success": false, "message": "유효하지 않은 Google ID Token입니다.", "data": null }
```

**실패 케이스 - 500 Internal Server Error**
```json
{ "success": false, "message": "서버 내부 오류가 발생했습니다.", "data": null }
```

## 📍 03 토큰 재발급 API

- **설명:** 만료된 AccessToken을 RefreshToken을 사용하여 재발급하는 API입니다. RefreshToken은 Redis에 저장되며, 유효한 경우 새로운 AccessToken과 RefreshToken을 동시에 발급합니다. (Token Rotation)
- **헤더:** 없음

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| refreshToken | 필수 | String | 기존에 발급받은 RefreshToken |

**Request Sample**

```json
{
  "refreshToken": "eyJhbGciOiJIUzI1NiJ9..."
}
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | AccessToken 및 RefreshToken 재발급 완료 |
| 실패 | 400 | refreshToken 누락 |
| 실패 | 401 | 유효하지 않거나 만료된 RefreshToken |
| 실패 | 500 | 서버 내부 오류 |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| accessToken | 필수 | String | 새로 발급된 JWT AccessToken |
| refreshToken | 필수 | String | 새로 발급된 RefreshToken (기존 토큰 무효화) |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "API 호출 성공",
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiJ9...(new)",
    "refreshToken": "eyJhbGciOiJIUzI1NiJ9...(new)"
  }
}
```

**실패 케이스 - 400 Bad Request**
```json
{ "success": false, "message": "요청 형식이 올바르지 않습니다.", "data": null }
```

**실패 케이스 - 401 Unauthorized**
```json
{ "success": false, "message": "인증 정보가 유효하지 않습니다. 다시 로그인해주세요.", "data": null }
```

**실패 케이스 - 500 Internal Server Error**
```json
{ "success": false, "message": "서버 내부 오류가 발생했습니다.", "data": null }
```

## 📍 05 이용약관 가져오기 API

- **설명:** 앱 내 이용약관 및 개인정보 처리방침 내용을 조회하는 API입니다. 로그인 전 단계에서도 접근 가능합니다.
- **헤더:** 없음

**Parameter**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| type | 필수 | String | 약관 유형 (TERMS: 이용약관, PRIVACY: 개인정보 처리방침) |

**Request Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |

**Request Sample**

```
GET /api/auth/terms?type=TERMS
```

**응답코드**

| 성공 여부 | 응답 코드 | 설명 |
| --- | --- | --- |
| 성공 | 200 | 약관 내용 조회 성공 |
| 실패 | 400 | 잘못된 type 값 |
| 실패 | 500 | 서버 내부 오류 |

**Response Body**

| 필드 이름 | 필수 여부 | 타입 | 설명 |
| --- | --- | --- | --- |
| type | 필수 | String | 약관 유형 |
| content | 필수 | String | 약관 본문 내용 |
| updatedAt | 필수 | DateTime | 약관 최종 수정일 (ISO 8601) |

**JSON 형식 응답**

```json
{
  "success": true,
  "message": "API 호출 성공",
  "data": {
    "type": "TERMS",
    "content": "제 1조 (목적) 이 약관은...",
    "updatedAt": "2025-06-01T00:00:00Z"
  }
}
```

**실패 케이스 - 400 Bad Request**
```json
{ "success": false, "message": "요청 형식이 올바르지 않습니다.", "data": null }
```
